Detailed Description

Modul untuk menyimpan dan memperbarui data status robot sepak bola (Kiper, Back, Striker).

Modul ini menerima data dalam bentuk array 18-byte dari masing-masing robot, lalu menyimpannya dalam variabel global untuk digunakan oleh simulasi, strategi, dan visualisasi.

Fungsi utama:

  • robotKiper(data, address)
  • robotBack(data, address)
  • robotStriker(data, address)

Setiap fungsi akan mem-parsing data yang dikirim robot dan menyimpannya ke dalam struktur data global berdasarkan ID robot.
