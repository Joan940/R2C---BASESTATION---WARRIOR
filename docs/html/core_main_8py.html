Detailed Description

Modul utama visualisasi simulasi sepak bola robot menggunakan pygame.

Menyediakan fungsi untuk menggambar berbagai elemen seperti robot, bola, musuh, heading, dummy, dan grid. Juga terdapat fungsi perhitungan geometri untuk heading, posisi bola, dan jarak antar objek.

Modul ini digunakan untuk memvisualisasikan data robot secara real-time dalam tampilan lapangan.
