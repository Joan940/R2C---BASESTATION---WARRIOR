Menampilkan teks ke layar menggunakan font kustom dari file TTF.

Fungsi ini merender teks menggunakan font Consolas dari path lokal. Teks akan ditampilkan terpusat di titik (x, y) pada layar.

Parameters
screenObjek surface Pygame tempat teks akan ditampilkan.
textString atau angka yang akan ditampilkan.
xKoordinat X posisi tengah teks.
yKoordinat Y posisi tengah teks.
sizeUkuran font (default: 10).
colorWarna font (default: diambil dari colorlist berdasarkan teks).
font_typeObjek font alternatif (default: font dari file .ttf lokal).
Returns
pygame.Rect Area tempat teks dirender.
Note
  • Jika parameter color tidak diberikan, warna diambil dari dictionary cc.colorlist.
  • Jika font_type tidak diberikan, font default adalah "Consolas.ttf" dari folder lokal.
