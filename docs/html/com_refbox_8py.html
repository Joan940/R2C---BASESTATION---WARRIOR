Detailed Description

Modul komunikasi TCP dengan Refbox (Referee Box) untuk simulasi robot sepak bola.

Modul ini bertanggung jawab untuk membuat koneksi ke Refbox menggunakan socket TCP, menerima dan memproses perintah pertandingan baik dalam bentuk string biasa maupun format JSON. Hasil komunikasi digunakan untuk mengatur status pertandingan di varGlobals.

Modul ini juga mengatur identifikasi tim, start dan stop pertandingan, serta event-event lain seperti kickoff, goal, penalty, dan sebagainya.
