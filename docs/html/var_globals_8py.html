Detailed Description

Modul variabel global dan konfigurasi untuk simulasi robot sepak bola.

Modul ini menginisialisasi jendela Pygame, memuat aset grafis, dan mendefinisikan variabel global seperti koordinat bola, status robot, mode permainan, serta input keyboard.

Semua file lain mengakses variabel dari modul ini untuk menjaga konsistensi dan koordinasi antar fungsi dalam sistem simulasi dan strategi.
