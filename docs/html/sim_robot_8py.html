Detailed Description

Modul simulasi pengiriman data ke robot (Kiper, Back, Striker) melalui UDP multicast.

File ini menyediakan fungsi untuk membentuk dan mengirim paket data ke masing-masing robot berdasarkan ID dan perintah tertentu. Modul ini digunakan dalam lingkungan simulasi sepak bola robot.

Format data dikirim dalam bentuk bytearray(18) dan mencakup informasi posisi, arah, status bola, dll.
